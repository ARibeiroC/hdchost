# hdchost

#### Site de Hospedagem web desenvolvido para treinamento das habilidades básicas que um programador front-end necessita saber para sua carreira profissional.

## Stacks
Projeto criado com HTML5 - Semantic, CSS3 e Javascript Vanilla.
Para este projeto eu utilizei as regras novas para CSS3 com Nesting (Encadeamento) trazendo uma maior clareza dos estilos e leitura do código, também trabalhei a parte semantica do HTML, para que isso facilite as ferramentas de acessibilidade, ao invés de utilizar Aria, para trabalhar com acessibilidade.

link: https://hdchosti.netlify.app/
